Acerca del sistema

Este sistema ha sido diseñado para facilitar la gestión operativa de Supermercados Líder. Desde este panel podrá acceder a los módulos de Inventario, Ventas, Proveedores, Clientes, Productos y Ofertas. Cada módulo está optimizado para brindar claridad, rapidez y control.

Si usted es Gerente, también tendrá acceso al Panel de Control, donde podrá visualizar indicadores estratégicos y gráficos integrados desde Power BI para apoyar la toma de decisiones.
